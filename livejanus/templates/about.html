About LiveJanus

LiveJanus enables multiple users to simultaneously maintain a census. It is free to use; if you are curious, sign up and give it a spin.

A user, once signed up, can create an event from their user page. Click on the new event in the user page to view and change the event details. From the event details page, the user can create sub-accounts for the census users. These users are created with a default password, but these passwords can quickly be changed from the event details page. Basic events can have a maximum of 2 census users, and do not save granular data.

Census users can log in to an event using the event key (found on the event details page), along with their username and password (also found and configured on the event details page). The event key can be shared via the event URL. Once logged in, census users can see and update a live tally (see example below).

A premium event allows you to assign an unlimited number of users to an event, download timestamped data, and set a maximum value to the counter. Premium events are purchased on a per-event basis, and are provided without guarantee or warranty. Payment is processed via Stripe.
